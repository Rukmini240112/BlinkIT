📊 BlinkIT Grocery Data Analysis – Power BI Project
📌 Overview

This project focuses on analyzing BlinkIT grocery data using Power BI to extract meaningful insights and support data-driven decision-making. The dashboard provides a clear view of sales performance, product trends, and outlet efficiency.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Compare outlet types and locations
Understand trends and patterns in the data

🛠️ Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
📈 Key Features
Interactive dashboard with filters and slicers
KPI indicators for quick insights
Visualizations like bar charts, line charts, and pie charts
Drill-down functionality for detailed analysis

🔍 Insights
Identified top-performing product categories
Analyzed sales trends across different outlets
Compared performance based on location and outlet size
Highlighted areas for business improvement
📂 Dataset

The dataset contains grocery sales data including product details, outlet information, and sales metrics.

🚀 Conclusion

This project demonstrates how raw data can be transformed into meaningful insights using Power BI, helping businesses make informed decisions and improve performance.
